# Dangle Diaries Store

**Dangle Diaries** is a student e-commerce project showcasing a premium yet affordable earrings store.  
The website is designed to allow customers to view products and order via WhatsApp.

## Features
- Hero section with store tagline
- Product showcase with sample products
- Responsive design for mobile and desktop
- Footer with contact and social media links
